# Token 管理器 GUI

轻量级 Factory Token 管理工具，基于 tkinter 构建。

## 功能特性

- 管理多个 Factory 账号的 Token
- 自动刷新 Token 并查询额度
- 支持账号快速切换
- 后台监控额度（每 90 秒检查一次）
- 额度用完自动提醒

## 文件说明

| 文件 | 说明 |
|------|------|
| `~/.factory/auth.json` | 当前激活的账号（Factory 客户端使用） |
| `tokens.json` | 备用账号池（程序目录下） |

## 按钮功能

**当前激活区域**
| 按钮 | 功能 |
|------|------|
| 检查额度 | 查询当前账号的额度使用情况 |
| 开始监控 | 每 90 秒自动检查额度，用完时弹窗提醒 |

**备用账号操作**
| 按钮 | 功能 |
|------|------|
| 刷新列表 | 重新加载备用账号列表 |
| 检查全部 | 一键查询所有备用账号额度 |
| 切换选中 | 将选中账号设为当前使用，原账号移入备用池 |
| 导入Token | 批量导入账号 |
| 删除选中 | 删除选中的备用账号 |

## 导入格式

每行一条，格式：
```
refresh_token----access_token----时间戳
```

## 使用说明

1. 启动程序后，会自动读取 `~/.factory/auth.json` 中的当前账号
2. 点击「导入Token」添加备用账号到账号池
3. 点击「检查全部」查看所有备用账号额度
4. 选中一个账号，点击「切换选中」即可切换
5. 建议开启监控，额度用完会自动提醒

## 注意事项

- 只有额度 100% 用完的账号才无法切换
- 关闭程序时，当前账号会自动保存回备用池
- 关闭程序后 `auth.json` 会被清空，需重新切换账号

## 运行

```bash
python token_manager_gui.py
```

## 依赖

- Python 3.10+
- requests
